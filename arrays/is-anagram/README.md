# Valid Anagram

[NeetCode Problem Link](https://neetcode.io/problems/is-anagram?list=neetcode150)

## Problem
Given two strings s and t, return true if t is an anagram of s, and false otherwise.

## Example
Input: s = "anagram", t = "nagaram"
Output: true

Input: s = "rat", t = "car"
Output: false
